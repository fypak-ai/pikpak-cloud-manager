⛅ PikPak Cloud Manager

Gerencie sua nuvem PikPak localmente — extraia links de compartilhamentos, navegue seus arquivos e transfira para o Dropbox.

Python 3 + Flask Dark Mode Auto-Login Dropbox Transfer Open Source
⭳ Download ZIP 📄 Ver no GitHub
🔗

Extractor de Links

Cole uma URL de compartilhamento PikPak e obtenha links de download direto com validade de ~24h.

☁

Minha Nuvem

Navegue suas pastas do PikPak via browser — breadcrumb, ícones por tipo, seleção múltipla de arquivos.

📦

Transfer para Dropbox

Envie arquivos selecionados diretamente do PikPak para o Dropbox com barra de progresso em tempo real.

🔒

Auto-Login Seguro

Token salvo localmente — login automático nas próximas sessões. Suporte a captcha integrado.

⚙ Como usar

# 1. Clone o repositório
git clone https://github.com/fypak-ai/pikpak-cloud-manager.git
cd pikpak-cloud-manager

# 2. Instale as dependências
pip install flask requests

# 3. Execute
python pikpak_extractor.py

# Acesse em http://localhost:5000
