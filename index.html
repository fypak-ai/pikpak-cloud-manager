PikPak Cloud Manager

Gerencie seus arquivos PikPak direto do navegador

Python + Flask Windows Dark Mode Dropbox

📦 Instalação

Siga os passos abaixo no PowerShell do Windows.

Passo 1 — Requisito

Instale o Python 3.10+

Baixe em python.org/downloads e marque "Add Python to PATH" durante a instalação.

Passo 2 — Criar pasta e baixar arquivos

Abra o PowerShell e rode:

mkdir C:\PikPakManager
cd C:\PikPakManager
Invoke-WebRequest -Uri "https://raw.githubusercontent.com/fypak-ai/pikpak-cloud-manager/main/pikpak_extractor.py" -OutFile "pikpak_extractor.py"
Invoke-WebRequest -Uri "https://raw.githubusercontent.com/fypak-ai/pikpak-cloud-manager/main/v7_template.html" -OutFile "v7_template.html"
Passo 3 — Instalar dependências
pip install flask requests
Passo 4 — Iniciar o app
python pikpak_extractor.py

Acesse no navegador: http://localhost:5000

🔄 Atualizar

cd C:\PikPakManager
Invoke-WebRequest -Uri "https://raw.githubusercontent.com/fypak-ai/pikpak-cloud-manager/main/pikpak_extractor.py" -OutFile "pikpak_extractor.py"

python pikpak_extractor.py

🚀 Funcionalidades

🔗 Link Compartilhado

Cole uma URL mypikpak.com/s/... e extraia todos os links de download — incluindo arquivos únicos com conteúdo magnet.

☁️ Minha Nuvem

Navegue pelos seus arquivos PikPak com login via Access Token. Suporte a breadcrumb e seleção múltipla.

📤 Envio para Dropbox

Selecione arquivos e envie em massa para o Dropbox com barra de progresso.

🔑 Como obter o Access Token

1. Abra o PikPak Web → pressione F12 → aba Network
2. Filtre por api/v1 e copie o header Authorization: Bearer XXX
3. Cole o token no app
