Instalação

Rodando em 4 passos

Só precisa do Python instalado. Nenhuma configuração complicada.

1

Instale o Python

Baixe em python.org e marque “Add Python to PATH” durante a instalação.

2

Baixe os arquivos

Salve os dois arquivos em C:\PikPakManager

curl.exe -o pikpak_extractor.py https://raw.githubusercontent.com/fypak-ai/pikpak-cloud-manager/main/pikpak_extractor.py
3

Instale dependências

Abra o terminal na pasta e rode:

pip install flask requests
4

Inicie o app

Execute e acesse em localhost:5000

python pikpak_extractor.py
Dica

Como obter o Access Token

O login via token é mais estável que email/senha — sem captcha, sem bloqueio.

🔑

Extraindo pelo DevTools

Abra mypikpak.com → F12 → aba Network → clique em qualquer pasta → encontre um request com api-drive → copie o header Authorization: Bearer eyJ...

💾

Token salvo automaticamente

Após o primeiro login, o app salva o refresh_token em pikpak_tokens.json. Na próxima vez, reconecta sozinho sem precisar colar o token de novo.
